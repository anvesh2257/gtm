Frequently Asked Questions

How does the dataLayer tracking work here? +
Custom JS binds event listeners to DOM elements (tabs, accordions). Upon triggering, it maps to a standard `trackEvent` function which pushes directly to the `window.dataLayer`.
What is Omnibug? +
Omnibug is a browser extension that decodes and displays outgoing analytics requests (like GA4 and GTM) in your Developer Tools panel. It makes debugging events significantly easier.
Where is the GTM Container ID? +
Open `js/tracker.js` and edit the `GTM_ID` constant to replace `GTM-XXXXXXX` with your actual tracking container ID.
